# MailZero

A satirical SaaS marketing page for a fictional service that creates Carbon
Avoidance Credits by promising to print political mail and then not printing,
mailing, delivering, or annoying anyone with it.

## GitHub Pages

This is a static site with no build step.

1. Push these files to a GitHub repository.
2. In the repository settings, enable GitHub Pages.
3. Set the source to the repository root.

The entry point is `index.html`.
